# 🎨 Project 1 - Nexus Studio

## Full Stack Project 1 - DecodeLabs 2026

---

## 📖 Description

Landing page responsive pour une agence digitale fictive **"Nexus Studio"**.

Projet réalisé dans le cadre du **Full Stack Project 1** de **DecodeLabs** (Promotion 2026).

L'objectif était de construire une expérience numérique **Mobile-First** en utilisant uniquement **HTML, CSS et JavaScript Vanilla**, sans framework ni bibliothèque externe (hors FontAwesome pour les icônes).

---

## 🛠️ Technologies utilisées

| Technologie | Utilisation |
|-------------|-------------|
| **HTML5** | Structure sémantique de la page (header, nav, main, article, footer) |
| **CSS3** | Design responsive (Grid, Flexbox, Mobile-First, Container Queries) |
| **JavaScript Vanilla** | Interactivité (menu burger, formulaire, animations, retour en haut) |
| **FontAwesome** | Icônes vectorielles |
| **Git / GitHub** | Versionnement et déploiement (GitHub Pages) |

---

## ✨ Fonctionnalités

- ✅ **Design responsive (Mobile-First)**
- ✅ **Menu burger** pour mobile
- ✅ **Formulaire de contact** avec validation en JavaScript
- ✅ **Animations au scroll** (apparition des cartes)
- ✅ **Bouton retour en haut**
- ✅ **Accessibilité (WCAG)** : balises sémantiques, aria-label, alt sur les images
- ✅ **Images avec nom et description** dans les projets
